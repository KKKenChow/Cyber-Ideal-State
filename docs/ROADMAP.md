# Cyber-Ideal-State Roadmap

本文档概述了 Cyber-Ideal-State（赛博理想国）的发展路线图。

---

## v0.0.1 - MVP（当前）

### 核心功能

- ✅ 基础数据模型（Role / Session / Decision / Persona / Memory）
- ✅ 数据采集器（微信✅、QQ TXT✅、飞书🔄认证可用、邮件✅、照片✅）
- ✅ Agent 生成引擎（采集 → 分析 → 生成 SOUL.md → 注册）
- ✅ 角色管理（CRUD、筛选）
- ✅ 会话管理（CRUD、消息管理、会话暂停/恢复）
- ✅ 会话引擎（自由讨论、轮流发言、辩论、投票、共识）
- ✅ 决策引擎（加权投票✅、多轮辩论✅、共识协商✅、权重多样性归一化✅）
- ✅ 三等级会话制（统治者/护卫者/劳动者 + 权限矩阵 + 理想国模式识别）
- ✅ Web UI（React + TypeScript + Tailwind CSS + FastAPI）
- ✅ OpenClaw 集成（CLI 调用、配置同步、API Key 检测）
- ✅ 一键安装/启动/卸载脚本

### 已知限制

- 角色编辑功能未实现（UI 按钮已预留但 disabled）
- 角色导入导出仅有后端方法，无 API 端点和 UI 入口
- 飞书采集器认证可用但消息/文档采集核心方法为 TODO
- QQ MHT 格式解析未实现
- 无 OpenClaw 时决策引擎返回占位数据（投票全 YES、共识全 agree）
- `permissions.yaml` 中 `can_chat_with` / `can_mention` 字段声明但未执行

## 版本策略

- **主版本号**（Major）：重大架构变更，可能不兼容
- **次版本号**（Minor）：新功能，向后兼容
- **修订号**（Patch）：Bug 修复，小改进

### 发布周期

- **Alpha 版本**：内部测试，功能不完整
- **Beta 版本**：公开测试，功能基本完整
- **RC 版本**：候选发布，准备正式版
- **正式版**：稳定发布，生产可用

路线图会根据社区反馈和技术发展进行调整。
